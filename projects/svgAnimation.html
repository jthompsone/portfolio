SVG Animations

Inspired by Valve's Portal series

Adding Animation

Found online, I took these Scalable Vector Graphics (SVG) and modified the layers to make it easier to animate -- such as reorganizing layers and renaming them to be recognizable, then applied CSS animation properties to each portion individually, along with some recoloring to give a more unique final image.

The animation properties are identical to the personal web animation projects I did in Test 00 and Pirate's Cove.
